# DESIGN.md

## 气质与意象
深夜音乐制作工作室——暖色 VU 表灯光映照在暗色调音台上，声音在暗空间中被赋予视觉形态。产品核心是"把看不见的声音变成看得见的画面"，操作流线如调音推子般干脆利落。

## 视觉策略
- 暗色底+暖色高光为主调，所有信息层级通过亮度和色温区分
- 音频波形作为装饰性视觉元素（转换进行时脉动）
- 卡片式布局，面板间有呼吸感间距

## 配色方案
| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 页面背景 | #0a0a0f | 关灯后录音室墙壁 |
| 卡片/面板 | #14141f | 调音台面板 |
| 主强调 | #e8a023 | VU 表琥珀灯光 |
| 成功 | #22c55e | 录音就绪绿灯 |
| 错误 | #ef4444 | 削波红灯 |
| 文字 | #e8e8ed | 控制室显示屏 |
| 辅助文字 | #6b6b80 | 面板丝印标注 |

## 字体排版
- 系统字体栈（PingFang SC / Microsoft YaHei / system-ui），无需额外加载
- 标题 600 weight，正文 400；紧凑行高，功能界面节奏

## 动效与交互
- 进度条：琥珀渐变填充，转换中带微光脉动
- 拖拽区：hover 时边框渐亮、微缩放
- 状态切换：300ms ease-out 过渡
- 文件卡片进入：轻微上移 + fade-in

## 页面结构
1. 顶部标题区：应用名 + 一句话说明
2. 上传区：左右双卡——MP3 拖拽上传 + 封面图上传（含预览）
3. 文件列表区：每条文件名 / 大小 / 状态进度 / 操作按钮
4. 底部操作栏：开始转换 / 下载全部 / 清空列表

## 设计禁忌
- 禁止冷色调蓝紫渐变作为主色
- 禁止浅色背景
- 禁止超大圆角（卡片 ≤ 16px）
- 禁止无状态反馈的空操作（点击必须有视觉回应）
